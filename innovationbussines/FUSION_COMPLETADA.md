## 🚀 INICIO RÁPIDO

### Windows
Ejecuta el archivo batch:
```bash
.\INICIAR-SISTEMA.bat
```

### Manual
**Terminal 1 - Backend:**
```bash
cd backend
npm install
npm run init-db
npm run dev
```

**Terminal 2 - Frontend:**
```bash
cd frontend
npm install
npm run dev
```
